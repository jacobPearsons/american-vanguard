Project Name:

American Vanguard Institute.

Core Features:

    Course registration and scheduling

    Student and staff directories

    Campus announcements and updates

    Academic notifications and alerts

    Assignment submission portals

    Department collaboration spaces

User Roles:

    Student

    Staff/Faculty

    Department Admin

    University Admin

Frontend Requirements:

    Student dashboard (grades, schedule, notifications)

    Staff dashboard (courses, advisees, department tasks)

    Course management interface

    Department collaboration hub

    Profile and directory views

Backend Requirements:

    Course registration API

    User directory API

    Announcements and alerts service

    Assignment management API

    Notification delivery system

Database Strategy:

    Relational database for students, staff, courses, and enrollments

    Document store for activity logs, announcements, and submissions

Authentication Method:

    Session authentication with university credentials

Deployment Target:

    Cloud container platform (AWS/GCP/Azure)

Scaling Expectations:

    10k concurrent users (students and staff during peak registration periods)
